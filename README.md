### Setup .env file
-Backend
```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```

### Build the app

-Frontend
```shell
npm i
```

-Backend
```shell
npm i
```
### Start the app

-Frontend
```shell
npm start
```

-Backend
```shell
npm start
```
